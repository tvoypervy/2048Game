# 2048
  2048 is a popular puzzle game where players move tiles on a 4×4 grid, merging identical numbers. The goal is to create a tile with the number 2048 while improving your strategic thinking
  and planning skills.
  
# Live Demo
  - [**2048**](https://tvoypervy.github.io/2048Game/)

# Technologies Used
  - **HTML** — page structure
  - **CSS** — styling and layout
  - **JavaScript** — game logic and interactivity

# Game Features
  - **Merge tiles with identical numbers**
  - **Track your current score**
  - **Control the game using arrow keys**
  - **Start a new game anytime**
